Simple and lightweight plugin for inserting custom javascript code for specific posts or pages.

== Description ==

This plugin provides an option to insert custom javascript code for each post or page separately.

### Features
* Light weight
* User-friendly
* No configuration required
* Insert js directly or from a source


== Installation ==

### Installation Instructions:

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to edit any post or page, scroll down and you will see a section titled "Custom JS" - there you can enter code directly or source URLs that you want to use for that page or post.
